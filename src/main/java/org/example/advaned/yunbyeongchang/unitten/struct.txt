호환되지 않는 두 객체를 큰 수정을 가하지 않고 연결하는 방식
ex) DVI 단자 관련 interface를 사용하던 중 HDMI 단자 클래스가 등장했을 때
HDMI 단자 전용 인터페이스를 추가하면 효율이 떨어짐(기능 자체는 화면 출력으로 동일한 부분이 존재하기 때문)
-> VideoAdapter 클래스를 추가하여 DVI interface를 참조
 이후 VideoAdapter에 HDMI 정보를 넣은 후, @Override 된 함수를 통해 연결
 세부 기능은 다르지만 화면을 켠다는 동일한 기능(명명)의 함수를 같이 사용할 수 있게 된다.
